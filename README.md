## Assignment Overview

This assignment demonstrates the configuration and implementation of various routing techniques using Cisco Packet Tracer. The exercises cover router setup, static routing, and dynamic routing protocols.

### Exercise 1 – Router Initial Setup

The following configurations were completed:

- Router hostname configuration
- Console access security
- User authentication and login settings

### Exercise 2 – Static Routing Implementation

Tasks completed:

- Network IP addressing configuration
- Manual route configuration using static routes
- Verification of network communication between devices

### Exercise 3 – RIP Routing Configuration

Tasks completed:

- Configuration of RIP Version 2
- Automatic route exchange between routers
- End-to-end connectivity validation

### Exercise 4 – EIGRP Routing Configuration

Tasks completed:

- EIGRP configuration with Autonomous System (AS) 100
- Establishment of EIGRP neighbor adjacencies
- Network connectivity testing and verification
